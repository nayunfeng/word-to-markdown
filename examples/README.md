# 示例目录说明

这个目录用于放测试用 `.docx` 文档。

推荐准备以下几类样例：

## 1. 标准标题文档
包含：
- Heading 1
- Heading 2
- Heading 3

用于验证标题拆分是否正常。

## 2. 非标准标题文档
包含：
- 第一章
- 一、
- （一）
- 1.
- 1.1
- （1）

用于验证启发式标题识别。

## 3. 表格文档
包含：
- 简单二维表格
- 多列表格
- 表格内换行

用于验证 Markdown 表格输出。

## 4. 图片文档
包含：
- 单张图片
- 多张图片
- 图文混排

用于验证图片抽取和图片插入位置。

## 5. 综合文档
建议包含：
- 标题
- 段落
- 列表
- 表格
- 图片

这是最接近生产环境的测试样例。

## 推荐命名

```text
examples/
├─ 01-standard-headings.docx
├─ 02-custom-headings.docx
├─ 03-tables.docx
├─ 04-images.docx
└─ 05-mixed.docx
```
